About Indeed:¶

In this workbook, the aim is to identify the market trends and required skills regarding jobs in Data Scientist/Analysis/Engineering.¶

PowerBI¶

Power BI is a collection of software services, apps, and connectors that work together to turn your unrelated sources of data into coherent, visually immersive, and interactive insights. Your data may be an Excel spreadsheet, or a collection of cloud-based and on-premises hybrid data warehouses. Power BI lets you easily connect to your data sources, visualize and discover what's important, and share that with anyone or everyone you want.

Tableu¶

Tableau is a powerful and fastest growing data visualization tool used in the Business Intelligence Industry. It helps in simplifying raw data into the very easily understandable format. Data analysis is very fast with Tableau and the visualizations created are in the form of dashboards and worksheets. The data that is created using Tableau can be understood by professional at any level in an organization. It even allows a non-technical user to create a customized dashboard.The great thing about Tableau software is that it doesn't require any technical or any kind of programming skills to operate.

from wordcloud import WordCloud, STOPWORDS 
import matplotlib.pyplot as plt 
import pandas as pd 

df = pd.read_csv('indeed_job_dataset.csv')

comment_words = '' 
stopwords = set(STOPWORDS) 

for word in df.Skill: 
    word = str(word) 
    tokens = word.split()  
    for i in range(len(tokens)): 
        tokens[i] = tokens[i].lower() 
    
    comment_words += " ".join(tokens)+" "

wordcloud = WordCloud(width=1600,height=1000, max_words=20,
                relative_scaling=0.5,
                background_color="white", 
                stopwords = stopwords,
                min_font_size = 10,
                collocations=False).generate(comment_words) 


plt.figure(figsize = (8, 8), facecolor = None) 
plt.imshow(wordcloud) 
plt.axis("off") 
plt.tight_layout(pad = 0) 

plt.show()

Conclusion:¶

1) Python is still the dominant programming language of choice but R is gaining popularity rapidly even though the first stable beta release of R was back in 2000 during which python had already released its second version the Python 2.0.

2) Oracle is the most popular Database Management System worldwide as of June 2020. Oracle also recently announced its Oracle Cloud Data Science Platform with features like: a) Python support for machine learning algorithms within the Oracle Autonomous Database that reduces the need to move data. b) New data catalog, to discover, find, organize, enrich and create data assets c) New big data service delivers a full Cloudera Hadoop implementation d) New service provides SQL access to HDFS and a new fully managed service to run Apache Spark applications

3) Tableau is one among the top Business Intelligence and Data Visualization tools available today. Most of the Fortune Global 500 companies are using this tool. In the Business Intelligence category, Tableau has a market spread of about 16.23% with 46,812 companies are using Tableau today. Source:Why is Tableu so popular

3) All the three cloud servce providers have their strengths and weakness. The choice of the cloud vendor is of the employer and since AWS is widely used by most companies, it has a high demand in the job market.

4) Business intelligence focuses on descriptive analytics, business analytics focuses on predictive analytics. Consider an E-Commerce website, from a business intelligence perspective one would look at how people are using your website, identifying trends in traffic, analyzing visitor demographics, and maybe even creating a system for tracking how customers click through different pages(descriptive analysis). A business analyst would deal more with the practical applications of this data and how it helps you make decisions for purchasing ads, creating new products, and updating your website. Source:Business Intelligence or Business Analytics

5) Apache Hadoop is an open-source framework written in Java that allows us to store and process Big Data in a distributed environment across various clusters of computers. For this Hadoop uses an algorithm called MapReduce, which divides the task into small parts and assigns them to a set of computers. Hadoop also has its own file system, Hadoop Distributed File System (HDFS), which is based on Google File System (GFS). HDFS is designed to run on low-cost hardware.

6) Apache Spark is an open-source distributed cluster-computing framework. Spark is a data processing engine developed to provide faster and easy-to-use analytics than Hadoop MapReduce. Hadoop and Spark can be used together, Spark brings speed and Hadoop brings one of the most scalable and cheap storage systems which makes them work together.
